# How to generate diagram

## Needed tools

- Java JRE (tested on openjdk)
- graphviz

## Folders/files presentation

- Here we store libs like plantuml.jar :
```
./lib
```

- Here we store our diagrams :
```
./out
```

- It's our script used to make diagrams. :
```
make.sh
```

## Here is our coolaid :

To generate diagrams :
```
./make.sh
```
